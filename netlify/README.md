# PDF Splitter - Netlify WebApp

A client-side PDF splitter that runs entirely in your browser. No server needed!

## 🚀 Deploy to Netlify

### Option 1: Drag & Drop (Easiest)

1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag and drop this entire folder
3. Your site is live instantly!

### Option 2: GitHub + Netlify

1. Push this folder to a GitHub repository
2. Go to [Netlify](https://app.netlify.com/)
3. Click "Add new site" → "Import an existing project"
4. Connect your GitHub repo
5. Click "Deploy site"

## ✨ Features

- 📤 Drag & drop PDF upload
- ⚡ Split up to 500 pages
- 📦 Download as ZIP
- 🔒 100% client-side (no data leaves your browser)
- 📱 Mobile friendly

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- [pdf-lib.js](https://pdf-lib.js.org/) - PDF manipulation
- [JSZip](https://stuk.github.io/jszip/) - ZIP creation
- [download.js](http://danml.com/download.html) - File downloads

## 📝 Custom Domain

After deployment, you can add a custom domain in Netlify settings.
